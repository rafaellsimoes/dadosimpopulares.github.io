Sobre Nós

O Dados Impopulares é um projeto acadêmico dedicado a disseminação de dados públicos. Nossa missão é transformar planilhas em Infográficos e Mapas temáticos.

  1. Facilitar a interpretação de dados públicos
  2. Produzir representações visuais para análise de informações
  3. Disseminar conhecimento
